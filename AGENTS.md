# CV Vincent Agents
Instructions for agents

## Project Brief
This project maintains the CV for the software developer Vincent Halasz written in Latex. Commands for generating pdf files and running latex interactively are contained in the Makefile.

## Structure
Directory structure of the repo

### cv_vincent
Contains the latex definitions of the cv

#### cv-sections
Contains the individual cv sections of the cv. Not all are used currently

#### fonts
Fonts for the cv

#### resume_cv.tex
Main latex file that imports the different sections

### output
Contains the output generated by commands like make pdf
